# Blog CRUD Application with User Authentication

## Description

A PHP and MySQL web application that allows users to register, log in, and manage blog posts using CRUD operations.

## Features

* User Registration
* User Login & Logout
* Password Hashing
* Session Management
* Create Posts
* View Posts
* Edit Posts
* Delete Posts

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* XAMPP
* Git
* GitHub

## Database

Database Name: blog

Tables:

1. users
2. posts
